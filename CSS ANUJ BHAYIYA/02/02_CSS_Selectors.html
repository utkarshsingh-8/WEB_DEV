Selectors in HTML

ELEMENT SELECTOR

the element selector HTML elements based on the element name

ID SELECTOR

The ID selector selects the id attribute of an HTML element to select a specific attribute .The id of an element is unique within a page , so the id selector is used to select one specific element!

CLASS SELECTOR

The class selectors selects an HTML elements with a specific class attribute.To select with a specific class ,write a period(.) character , followed by the class name
